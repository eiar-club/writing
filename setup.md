# Setup Win10

## WSL

- install wsl for windows (ubuntu 20)
- wsl into the subsystem

## Ubuntu 20

- ufw disable

## Install mkdocs

- apt-get install update
- apt install python-pip3
- pip3 install mkdocs
- pip3 install git+https://github.com/jldiaz/mkdocs-plugin-tags.git
- pip3 install mkdocs-material

## Mkdocs

- mkdocs serve
- mkdocs build
